# NewsApp - Jetpack Compose

A simple news application building with Jetpack Compose.  
This is an early-stage project, still under development.

## Tech's Used
- [Jetpack Compose]
- [Retrofit] 
- [NewsAPI]
- [Hilt]
- [MVVM Architecture]
